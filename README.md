# 浏览器多页面管理工具

一个智能的Chrome浏览器扩展，帮助你高效管理标签页和窗口，告别标签页混乱。

## 功能特性

### 🎯 核心功能

- **统一仪表盘**：集中展示所有浏览器窗口和标签页，支持拖拽移动、快速切换
- **多布局模式**：
  - 卡片网格：直观展示标签页详情
  - 紧凑列表：高信息密度，快速扫描
  - 杂志翻页：专注查看单个窗口
- **实时搜索**：按标题和URL实时过滤标签页，支持快捷键聚焦
- **重复页面检测**：自动识别重复页面，一键清理
- **高频页面推荐**：基于使用习惯智能推荐常用页面，一键直达
- **快捷操作**：全局快捷键、双击Command快速切换、方向键翻页

### ⌨️ 快捷键

| 功能 | Mac | Windows |
|------|-----|---------|
| 打开仪表盘 | `Control + D` | `Alt + Shift + D` |
| 关闭重复页面 | `Control + X` | `Alt + Shift + X` |
| 快速搜索标签页 | `Control + 1` | `Alt + 1` |
| 聚焦搜索框 | `⌘ + K` | `Ctrl + K` |

## 安装方法

### 开发者模式安装

1. 下载或克隆此仓库到本地
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目文件夹

## 使用说明

### 基本操作

1. **打开仪表盘**：
   - 点击扩展图标 → 点击「打开仪表盘」
   - 或使用快捷键 `Control + D` (Mac)

2. **搜索标签页**：
   - 在仪表盘或Popup中使用搜索框
   - 或使用快捷键 `Control + 1` 快速唤起搜索

3. **管理重复页面**：
   - 重复页面会以红色边框标识
   - 点击「清理重复」一键关闭所有重复页面

4. **切换布局**：
   - 在仪表盘顶部选择布局模式
   - 系统会自动记住你的偏好

### 高级技巧

- **双击Command**：在仪表盘内快速切换到随机标签页
- **方向键翻页**：杂志模式下使用左右方向键翻页
- **Alt/Option修饰键**：在搜索界面中，按住Alt键可强制跳转到原始窗口

## 技术栈

- **Chrome Extension API**：标签页管理、窗口管理、存储、通信
- **原生JavaScript**：无框架依赖，轻量高效
- **HTML5 + CSS3**：响应式界面设计

## 核心逻辑

### 标签页切换策略

根据项目设计，标签页切换遵循以下策略：

1. **同窗口目标**：直接切换到目标标签页
2. **跨窗口目标**：
   - 优先在当前窗口导航或打开新标签页，避免窗口跳跃
   - 若目标标签页属于Tab Group，则不关闭原始标签页
   - 按住Alt/Option键可强制跳转到原始窗口

### 高频页面统计

- 监听标签页创建事件，按URL维度累计打开次数
- 使用时间衰减算法，近期访问权重更高
- 自动过滤Chrome内部页面，合并相似URL
- 支持手动置顶和移除
